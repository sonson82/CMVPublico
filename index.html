PROYECTO FINAL INTEGRADOR

INTRODUCCIÓN

Tomé la decisión de desarrollar el proyecto final con Angular ya que no había tenido la posibilidad de tabajar con este framework antes.

La idea fue traspasar la barra de favoritos de mi navegador a una aplicación. Los datos para la gestión de los contenidos pueden ser extraídos de un fichero XML o de una BBDD, y se muestran en una galería de cartas. Al clicar sobre cada elemento, este voltea y nos muestra los enlaces relacionados en función de si son páginas oficiales, tutoriales o páginas para el análisis de ficheros.

Tiene tambien dos funciones extra: una búsqueda por nombre y una búsqueda mediante filtros.

CARACTERÍSTICAS

  • Framework: Angular
  • IDE: Visual Studio Code
  • Lenguaje de programación: Typescript y PHP (en servidor)
  • Estilos: CSS y Bootstrap
  • Conexión XML: XMLHttpRequest
  • Conexión BBDD: HttpClient (respuesta con Json)
  • Variables globales: RxJs (BehaviorSubject, Observable y Observer)
  • Proyecto: FULL-STACK FAVORITES
  • Código proyecto
  • Código servidor
  • Documentación BBDD
  • Fichero XML (generado manualmente)
